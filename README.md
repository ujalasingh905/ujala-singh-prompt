# VELOX – Transportation & Logistics Platform Frontend

VELOX is a modern and responsive transportation and logistics web application frontend designed for booking and managing transportation services such as trucks, taxis, tempos, autos, bikes, and vans.

The project focuses on delivering a smooth and visually engaging user experience with animated UI interactions, real-time tracking visualization, booking workflows, and responsive layouts suitable for modern transportation platforms.

---

# 🚀 Project Overview

VELOX provides a transportation booking interface where users can:

- Book transportation services
- Select different vehicle categories
- View estimated pricing
- Track delivery/vehicle movement visually
- View driver and delivery details
- Access pricing information
- Read customer reviews
- Contact support teams

The frontend is designed with:

- Modern dark-themed UI
- Responsive layouts
- Smooth animations
- Real-time inspired tracking interface
- Interactive booking forms
- Optimized user experience
